## Pipeline ETL wykorzystujący web scraping jako źródło danych


Celem projektu było pobranie i przetworzenie danych dotyczących 10 największych banków na świecie według kapitalizacji rynkowej. Na podstawie danych z zewnętrznego źródła stworzyłem pipeline ETL, który pozyskuje dane, przelicza kapitalizację na różne waluty (USD, GBP, EUR, INR) z użyciem dostarczonych kursów wymiany, a następnie zapisuje wyniki zarówno do pliku CSV, jak i do bazy SQL. Dodatkowo przygotowałem zestaw zapytań dla oddziałów w Londynie, Berlinie i New Delhi oraz system logowania postępu pracy.
